# Descendant selector ('ancestor descendant')

| jQuery | JS | Description |
|:--:|:--:|:--:|
| `$('ancestor descendant')` | `querySelectorAll('ancestor descendant')` | Selects all elements that are **_descendants_** of a given ancestor. |
