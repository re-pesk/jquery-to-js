# Child Selector (`parent > child`)

| jQuery | JS | Description |
|:--:|:--:|:--:|
| `$('parent > child')` | `querySelectorAll('parent > child')` | Selects all direct child elements specified by "**_child_**" of elements specified by "parent". |
