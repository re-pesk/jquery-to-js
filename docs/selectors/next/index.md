# Next Adjacent Selector ('prev + next')

| jQuery | JS | Description |
|:--:|:--:|:--:|
| `$('prev + next')` | `querySelectorAll('prev + next')` | Selects all **_next_** elements matching "next" that are **_immediately preceded_** by a sibling "prev" |
