| Utilities |||
|:--|:--|:--:|
| [$.grep(aray, (element?, index?) => {...}, invert?)](?grep/) | **_Finds the elements of an array_** which satisfy a filter function. The original array is not affected. | [API doc](https://api.jquery.com/jQuery.grep/) |
